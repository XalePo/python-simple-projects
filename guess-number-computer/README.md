## Guess the Number (Computer)

In this project, the user thinks of a number between 0 and 100, and the computer tries to guess it. After each guess, the user tells the computer whether the number was too high, too low, or correct. The program updates its lower and upper bounds based on that feedback and continues guessing until it finds the correct number. It also keeps track of how many attempts the computer needed. The project uses Python’s `random` module to generate guesses within the current bounds.
